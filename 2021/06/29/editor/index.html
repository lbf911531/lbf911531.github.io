模拟TextArea-踩坑记录

发表于 2021-06-29 更新于 2021-06-30 分类于 前端

思路提供

  1. 原生 div + textarea ，通过调整系列样式，将div设置为透明色，以textarea所在外层div包裹,相对外层div绝对定位。

  2. 使用div+contentEditable属性，将div设置为可编辑状态

    ps: 在使用contentEditable属性时，可以放心大胆使用。IE在这里是最先支持的，

属性 Google IE Firefox Safari Opera
contentEditable 11.0 5.5 3.0 3.2 10.6

实践

实践-‘思路一’

既然是踩坑，就先从第一条思路开始尝试

然后就会发现第一个问题： 光标看不见了！！光标默认跟随了编辑器内容颜色，因为 div[class=’inner’]文本颜色变成了白色，且背景色为白色，因此光标也是白色了。

解决：  ”caret-color “   // caret-color 属性用来定义插入光标（caret）的颜色
通过给 div[class='inner'] 设置caret-color: #000; 光标就又可以看见了

光标可见的问题虽然解决了，但随之出现了另一个问题：无法随意在文本中插入光标。

模拟onChange

在基本实现了模拟textarea的基础上，我们需要获取文本值。即类似于要模拟出表单控件都有的方法：onChange。

然而很遗憾： 走邪道总是崎岖的。HTML中contentEditable属性并不支持它。

事实上，在原生的HTML中，onChange仅支持 input, select,textarea( https://developer.mozilla.org/zh-CN/docs/Web/Events/input )

那么问题来了，onChange是文本的变化，文本在什么时候会变化？敲击键盘的时候（ps：还有复制粘贴，js强制插入文本；这里不做深入的兼容）。因此，键盘事件成了一条思路。我们可以监听keyup事件。【keydown,比起键弹起，键按下到弹起前的两种事件在这里一点都不合适，你不知道什么时候结束了敲击】

this.textArea.addEventListener("keyup", this.handleGetValue);

handleGetValue = e => {
  const { onChange } = this.props;
  const { innerText, innerHTML } = e.target;
  console.log(innerText, innerHTML);
}

回写文本

模拟了onChange之后，我们就能拿到文本值。同时抛出使用。这时候要想起一件事情——模拟的初衷在于高亮。高亮意味着，要对拿到的文本做关键字匹配，然后替换其为html片段，加入color样式。

然后让我们完善整合一下上面的代码：

componentDidMount() {
  const { highlightText } = this.props;
  this.setState({ highlightText })
  // keyup事件模拟input的onChange事件
  this.textArea.addEventListener("keyup", this.handleGetValue)
}  

componentWillReceiveProps(nextProps) {
  const { highlightText } = nextProps;
  this.setState({ highlightText })
}

// 等同onchange
handleGetValue = e => {
  const { onChange } = this.props;
  const { innerText, innerHTML } = e.target;
  if (onChange) {
    this.setState({
      cursorPos: posObj,
    }, () => {
      // innerText虽然输出在控制台，看不见值却又存在，有可能是 ‘↵ ’, 因此判断如果是该情况，给出一个 空串
      const string = !innerText || (innerHTML === '<br>') ? '' : innerText;
      onChange(string)
    })
  }
}

render() {
  const { highlightText } = this.state;

  return (
    <div style={{ position: 'relative', width: '100%', height: '125px' }}>
      <div
        style={{
          position: 'absolute',
          top: 0,
          right: 0,
          left: 0,
          bottom: 0,
          zIndex: 2,
          backgroundColor: 'transparent',
          padding: '4px 11px',
          outline: 'none',
          height: '125px',
          overflowY: 'auto',
          lineHeight: '1.5',
          verticalAlign: 'bottom',
        }}
        dangerouslySetInnerHTML={{ __html: highlightText }}
        contentEditable="true"
        id="define-textarea"
        ref={ref => { this.textArea = ref }}
      />
    </div>
  )
}

但这时候，如果你仔细一点，就会发现两个大坑！！

  • 第一： 光标永远在最开始的地方闪烁，而不是文本的末尾；

  • 第二，你不能输入中文了。

针对上诉二个问题，先看看原因。不难发现，是因为onChange回抛再重写，导致了光标始值被刷新在了开头，也是因为无时不刻的回抛重写，让事件以为我们意图将拼音算入了文本。

因此，对于第二问题：我们引入一个新东西： compositionstart，compositionend

compositionstart 事件触发于一段文字的输入之前（类似于 keydown 事件，但是该事件仅在若干可见字符的输入之前，而这些可见字符的输入可能需要一连串的键盘操作、语音识别或者点击输入法的备选词）。 详情参照 https://developer.mozilla.org/zh-CN/docs/Web/Events/compositionstart

通过 compositionstart，compositionend搭配使用，我们可以为onChange加一道锁，当发现进入了中文拼写时，锁置为true，直到拼音完成前的过程中，禁止掉onChange的回抛。

this.textArea.addEventListener("compositionstart", this.handleSetFlagTrue);
this.textArea.addEventListener("compositionend", this.handleSetFlagFalse);

handleSetFlagTrue = () => { this.flag = true; }

handleSetFlagFalse = () => { this.flag = false; }

再来就是第一个问题： 我们需要记录一下上一次onChange时光标的位置，然后在下一次onChange之前，将光标手动移到该位置上。这样就可以保证了光标不会回到最开始的地方闪烁。

// 获取光标位置
saveSelection = () => {
  const target = document.querySelector('#define-textarea');
  if (window.getSelection) {
    const range = window.getSelection().getRangeAt(0);
    const preSelectionRange = range.cloneRange();
    preSelectionRange.selectNodeContents(target);
    preSelectionRange.setEnd(range.startContainer, range.startOffset);
    const start = preSelectionRange.toString().length;

    return {
      start,
      end: start + range.toString().length,
    }
  } else if (document.body.createTextRange) {
    const selectedTextRange = document.selection.createRange();
    const preSelectionTextRange = document.body.createTextRange();
    preSelectionTextRange.moveToElementText(target);
    preSelectionTextRange.setEndPoint("EndToStart", selectedTextRange);
    const start = preSelectionTextRange.text.length;

    return {
      start,
      end: start + selectedTextRange.text.length,
    }
  }
};

同时这也能够让我们在文本中随意插入光标。

到这里为止，基本上，实现了div的contentEditable属性模拟textarea，并高亮的目的。

如果要延申到富文本编辑器的话，可以考虑 window.execCommond实现复制，粘贴，回退….

以下贴上完整代码（ps:毫无疑问，简单的模拟还有很多bug和需求都没实现，这些都将在后续文档中逐步补充完善）


import React from 'react';

class Editor extends React.Component {
  constructor(props) {
    super(props);
    this.state = {
      highlightText: null,
      cursorPos: 0,
    }
    this.flag = false;
    this.clock = true;
  }

  componentDidMount() {
    const { highlightText } = this.props;
    this.setState({ highlightText })
    // keyup事件模拟input的onChange事件
    this.textArea.addEventListener("keyup", this.handleGetValue)
    /**
     * compositionstart，compositionend
     * 是为了处理输入中文时，将拼音回写到了文本区中，监听二者事件加一个锁，
     * 在完成中文之前，不执行keyup的回调
     */
    this.textArea.addEventListener("compositionstart", this.handleSetFlagTrue)
    this.textArea.addEventListener("compositionend", this.handleSetFlagFalse)
  }

  componentWillReceiveProps(nextProps) {
    const { highlightText } = nextProps;
    this.setState({ highlightText }, () => {

      const { cursorPos } = this.state;
      this.restoreSelection(null, cursorPos);
    })
  }

  componentWillUnmount() {
    this.textArea.removeEventListener("keyup", this.handleGetValue)
    this.textArea.removeEventListener("compositionstart", this.handleSetFlagTrue)
    this.textArea.removeEventListener("compositionend", this.handleSetFlagFalse)
    this.setState = () => { }
  }

  // 等同onchange
  handleGetValue = e => {
    const { onChange } = this.props;
    const { innerText, innerHTML } = e.target;
    if (onChange && !this.flag) {
      const posObj = this.saveSelection();
      this.setState({
        cursorPos: posObj,
      }, () => {
        // innerText虽然输出在控制台，看不见值却又存在，有可能是 ‘↵ ’, 因此判断如果是该情况，给出一个 空串
        const string = !innerText || (innerHTML === '<br>') ? '' : innerText;
        onChange(string)
      })
    }
  }

  handleSetFlagTrue = () => { this.flag = true; }

  handleSetFlagFalse = () => { this.flag = false; }

  // 强制将光标挪动到文本最后
  keepLastIndex = () => {
    const target = document.querySelector("#define-textarea");
    if (window.getSelection) {// ie11 10 9 ff safari
      target.focus(); // 解决ff不获取焦点无法定位问题
      const range = window.getSelection();// 创建range
      range.selectAllChildren(target);// range 选择obj下所有子内容
      range.collapseToEnd();// 光标移至最后
    }
    else if (document.selection) {// ie10 9 8 7 6 5
      const range = document.selection.createRange();// 创建选择对象
      range.moveToElementText(target);// range定位到obj
      range.collapse(false);// 光标移至最后
      range.select();
    }

    this.setState({
      cursorPos: this.saveSelection(),
    });
  }

  /**
  * 设置光标位置
  * @param ctrl  需要设置对象
  * @param savedSel 对象中内容的光标的所在位置{start: ,end: }
  */
  restoreSelection = (target, savedSel) => {
    const ctrl = document.querySelector('#define-textarea')
    if (document.createRange) {
      let charIndex = 0;
      const range = document.createRange();
      range.setStart(ctrl, 0);
      range.collapse(true);
      const nodeStack = [ctrl];
      let node;
      let foundStart = false;
      let stop = false;

      while (!stop && (node = nodeStack.pop())) {
        if (node.nodeType === 3) {
          const nextCharIndex = charIndex + node.length;
          if (!foundStart && savedSel.start >= charIndex && savedSel.start <= nextCharIndex) {
            range.setStart(node, savedSel.start - charIndex);
            foundStart = true;
          }
          if (foundStart && savedSel.end >= charIndex && savedSel.end <= nextCharIndex) {
            range.setEnd(node, savedSel.end - charIndex);
            stop = true;
          }
          charIndex = nextCharIndex;
        } else {
          let i = node.childNodes.length;
          // eslint-disable-next-line no-plusplus
          while (i--) {
            nodeStack.push(node.childNodes[i]);
          }
        }
      }

      const sel = window.getSelection();
      sel.removeAllRanges();
      sel.addRange(range);
    } else if (document.body.createTextRange) {
      const range = document.body.createTextRange();
      range.moveToElementText(ctrl);
      range.collapse(true);
      range.moveEnd("character", savedSel.end);
      range.moveStart("character", savedSel.start);
      range.select();
    }
  };

  // 获取光标位置
  saveSelection = () => {
    const target = document.querySelector('#define-textarea');
    if (window.getSelection) {
      const range = window.getSelection().getRangeAt(0);
      const preSelectionRange = range.cloneRange();
      preSelectionRange.selectNodeContents(target);
      preSelectionRange.setEnd(range.startContainer, range.startOffset);
      const start = preSelectionRange.toString().length;

      return {
        start,
        end: start + range.toString().length,
      }
    } else if (document.body.createTextRange) {
      const selectedTextRange = document.selection.createRange();
      const preSelectionTextRange = document.body.createTextRange();
      preSelectionTextRange.moveToElementText(target);
      preSelectionTextRange.setEndPoint("EndToStart", selectedTextRange);
      const start = preSelectionTextRange.text.length;

      return {
        start,
        end: start + selectedTextRange.text.length,
      }
    }
  };

  // 插入文本
  insertText = (string,text) => {
    const { cursorPos } = this.state;
    if (cursorPos && string) {
      const stringContent = this.getContentFromHtml(string);
      const preStr = stringContent.substring(0,cursorPos.start);
      const lastStr = stringContent.substring(cursorPos.end,string.length);
      const finalStr = (`${preStr} ${text}${lastStr}`)
        .replace(/\s+,/g,',')
        .replace(/,\s+/g,',')
        .replace(/\(\s+/g,'(')
        .replace(/\s+\)/g,')'); // 过滤掉， （，）前后不必要的空格
      return finalStr
    } return text;
  }

    // 提取纯文本
    getContentFromHtml = value => {
      let target;
      if (typeof value === 'string') {
        target = value || '';
      } else {
        target = value.toHTML && value.toHTML() || '';
      }

      return (
        target ? target.replace(/<\/?[a-zA-Z]+.*?>/g, '') : ''
      )
    }

  render() {
    const { highlightText } = this.state;

    return (
      <div style={{ position: 'relative', width: '100%', height: '125px' }}>
        <div
          style={{
            position: 'absolute',
            top: 0,
            right: 0,
            left: 0,
            bottom: 0,
            zIndex: 2,
            backgroundColor: 'transparent',
            padding: '4px 11px',
            outline: 'none',
            height: '125px',
            overflowY: 'auto',
            lineHeight: '1.5',
            verticalAlign: 'bottom',
          }}
          dangerouslySetInnerHTML={{ __html: highlightText }}
          onClick={() => {
            const posObj = this.saveSelection();
            this.setState({
              cursorPos: posObj,
            }, () => {
              this.restoreSelection(null, posObj);
            })
          }}
          contentEditable="true"
          id="define-textarea"
          ref={ref => { this.textArea = ref }}
        />
      </div>
    )
  }
}

export default Editor
